Tsari: Precision Rhythms for Peak Performance
Tsari is a high-performance, minimalist productivity engine built as a Progressive Web App (PWA). Designed for those who view their daily routine as a system to be optimized, it merges schedule blocking, habit tracking, and focus tools into a single, sleek "Glassmorphism" interface.

🚀 Key Features
Dynamic Task Rotation: Automatically cycles through complex study modules or project categories (e.g., alternating between Thermodynamics and Python Dev) to maintain high cognitive engagement.

Embedded Focus Audio: A persistent, lightweight player for streaming Lofi or Deep Focus audio directly via the YouTube IFrame API without the friction of app-switching.

Integrated Precision Timer: A Pomodoro-style timer tied directly to your active blocks, allowing for granular tracking of actual "time-on-task."

PWA Architecture: Optimized for a native-app feel on iOS and Android. Fast, weightless, and accessible right from your home screen.

Local-First Privacy: No cloud databases or tracking. All data is stored securely and exclusively in your browser's localStorage.

🛠️ Tech Stack
Frontend: Vanilla JavaScript (ES6+), CSS3 (Custom Properties & Glassmorphism), HTML5.

Integrations: YouTube IFrame Player API for background audio.

Design: Inspired by Apple’s SF Pro design language for a tactile, premium user experience.
